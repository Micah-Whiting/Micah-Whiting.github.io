A website designed for fictional restaurant Pasta Amore. The goal of this project was to make a simple, artistic website that contained the restaurant's business hours, menu, bio, and reviews. The brief for this project dictated the inclusion of these pages as well as the use of the corporate brand style guide. Research was conducted throughout the creation process to find suitable imagery as well as a successful layout that met the expectations of the project manager. The final result was a clean, artistically stark website that met the project goals by providing meaningful content such as Pasta Amore's hours, telephone number, menu, biography, and reviews.

Website Created With: HTML, CSS.

Wireframes Created With: Adobe Illustrator.
